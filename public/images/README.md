# Image Replacement Guide

Replace these placeholder images with your own. Keep the same filenames or update the code accordingly.

## Required Images

### Hero Slider (/images/hero/)
- hero-1.jpg (1920x1080 or similar landscape, main hero image)
- hero-2.jpg (1920x1080, sustainability themed)
- hero-3.jpg (1920x1080, product showcase)

### Products (/images/products/)
For each product, you need:
- capri-1.jpg, capri-2.jpg, capri-3.jpg (product images, 800x1000 portrait)
- capri-blue.jpg, capri-coral.jpg, capri-black.jpg (color variants)
- riviera-1.jpg, riviera-2.jpg (and color variants)
- mykonos-top-1.jpg, mykonos-top-2.jpg (and variants)
- positano-1.jpg, positano-2.jpg (and variants)
- monaco-1.jpg, monaco-2.jpg (and variants)
- sardinia-1.jpg, sardinia-2.jpg (and variants)
- athens-1.jpg, athens-2.jpg (and variants)
- corsica-1.jpg, corsica-2.jpg (and variants)

### Collections (/images/collections/)
- mediterranean.jpg (800x1000, collection cover)
- cote-dazur.jpg (800x1000)
- santorini.jpg (800x1000)
- amalfi.jpg (800x1000)

### About (/images/about/)
- story.jpg (800x1000, founder/brand story)
- intro.jpg (800x1000, craftsmanship)
- sustainability.jpg (800x800, ocean/nature)

### Instagram (/images/instagram/)
- ig-1.jpg through ig-6.jpg (600x600 square images)

## Image Tips
1. Use high-quality JPG images (compress for web)
2. Recommended sizes are listed above but images will be responsive
3. For products, shoot on neutral backgrounds
4. Maintain consistent lighting across product shots
5. For best results, use images at least 2x the display size

## Using External Images
You can also use image URLs from services like:
- Unsplash (already configured in next.config.mjs)
- Cloudinary (already configured)
- Your own CDN

Just update the image paths in /src/lib/data/products.ts
